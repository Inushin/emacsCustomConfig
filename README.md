#Emacs CheatSheet

## Emacs

### Suspend Emacs

```emacs
Ctrl-z
```

### Exit permanently

```emacs
Ctrl-x Ctrl-c
```

### Advanced options

```emacs
Meta-x
```

## Files

### Open file

```emacs
Ctrl-x Ctrl-f
```

### Save changes

```emacs
Ctrl-x Ctrl-s
```

### Save changes on all open files

```emacs
Ctrl-x s
```

### Change currently open file for another in the same folder

```emacs
Ctrl-x Ctrl-v
```
